A tiny, self-contained Spring Boot service that turns HTML into PDF — including PDF/A (parts 1/2/3, levels A/B/U). Fully offline: no external services, no runtime configuration, all fonts and the sRGB profile bundled. The whole public contract is “POST HTML, get an application/pdf back”. Ships as a multi-arch (amd64 / arm64) Docker image.
